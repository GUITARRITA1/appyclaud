# AppyClaud

Página de bienvenida con botón de ingreso.

## 📁 Estructura del proyecto

- `index.html` - Página principal con botón "INGRESAR"
- `inicio.html` - Página de destino después del ingreso  
- `APPYC.JPG` - Imagen de fondo

## 🚀 Ver sitio web

El sitio estará disponible en: `https://GUITARRITA1.github.io/appyclaud/`

## 📝 Notas

- Asegurar que `APPYC.JPG` existe en la raíz del proyecto
- El botón "INGRESAR" redirige a `inicio.html`
